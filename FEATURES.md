# Feature Backlog

Potential features and improvements for the Health Dashboard, roughly ordered by impact.

## High Impact

### Period Comparison View
Compare metrics across time periods side-by-side (this month vs last month, this quarter vs previous). Lets users quickly see if they're improving beyond just the 30d trend percentage.

### Training Load / Fitness-Fatigue Model (ATL/CTL/TSB)
TRIMP-based acute/chronic training load chart using existing workout + HR + HRV data. Shows when you're peaking vs overreaching. Very valuable for anyone who trains seriously.

### Sleep Debt Tracker
Running cumulative sleep deficit against a configurable target (default 8h). A single number like "you owe your body 12 hours of sleep" is more actionable than nightly averages.

### PDF / Image Export
"Download Report" button that generates a shareable summary (key stats + mini charts) for a selected period. Useful for doctor visits or sharing with coaches.

## Medium Impact

### Medication / Supplement Log Overlay
Let users mark start/stop dates for supplements or medications. Show vertical reference lines on all charts for easy before/after analysis.

### Heart Rate Zone Analysis
Time-in-zone breakdowns (Z1-Z5) per workout and over time, using existing workout HR timeline data. Useful for endurance athletes.

### Weekly Digest View
Single-page structured "this week" summary: what went well, what declined, notable anomalies. Like AI Insights but scannable without an API call.

### Notifications / Thresholds
User-configurable visual alerts ("warn if HRV drops below 20ms", "resting HR above 70"). Since everything is local, these are just visual highlights on the dashboard.
